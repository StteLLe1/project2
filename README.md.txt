# Feedback Form Project

This project implements a simple feedback form that allows users to submit their name and a message. The submitted data is stored in the browser's localStorage.

## Installation

1. Clone the repository: `git clone https://github.com/ваш_пользователь/ваш_репозиторий.git`
2. Open `index.html` in your web browser.

## Usage

1. Fill out the form with your name and message.
2. Click the "Submit" button to save the feedback.
3. The feedback is stored in localStorage.

## Notes

- This project uses jQuery for DOM manipulation.
- The feedback data is stored in the browser's localStorage.
